HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • meta elements contain information about the page that is used by the browser.
  • The charset element sets our character encoding.
  • The http-equiv element sets an HTTP header for content.
  • The heading (h1, h2 etc) elements represent the level of heading a given text block represents.
  • The ul, ol, and li elements represent unordered lists, ordered lists, and list items.
  • The img element contains information about images that are displayed on the webpage.
  • The img element can contain various attributes; the src attribute is the most important because it defines the location of the image.
  • The br element creates a line of empty space, or a line break, between two blocks of content.
  • The a element (which stands for anchor) creates links to the same webpage or other webpages.
    The href attribute points to the URL for the link.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • The add command adds modifications in the current working branch to the staging area.
    The -A flag indicates that we want to add all changes.
  • The pull command is used to o receive a branch's modifications into the local environment.
  • The push command pushes the changes we have locally to our remote GitHub branch.
